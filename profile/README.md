## About
Shiryu Studios is an indie software studio focused on tools and services that support digital products, creator workflows, web platforms, and interactive experiences. This GitHub organization is the engineering home for that ecosystem, spanning public sites, internal systems, platform APIs, edge-ready templates, and game-related development work.

Rather than treating each property as a separate island, Shiryu is building a connected platform where brand, identity, operations, and product delivery can move as one system.

## Current Focus
- Building a connected web and cloud ecosystem across public, client, and internal surfaces
- Standardizing platform architecture around reusable templates, shared tooling, and API-first services
- Expanding identity and access foundations for authentication, account flows, and future SSO
- Exploring multiplayer and game-development infrastructure alongside web product delivery

## Selected Repositories
These repositories give the clearest snapshot of the organization’s public direction.

### [vite-react-template](https://github.com/Shiryu-Studios-LLC/vite-react-template)
A React + Vite + TypeScript starter built for **Cloudflare Workers** with **Hono**, designed for edge deployment, hot module replacement, API routing, and full-stack development from a single template.

### [BethanySDAWebsite](https://github.com/Shiryu-Studios-LLC/BethanySDAWebsite)
A production-style web project that includes a frontend app structure along with `functions/api`, `migrations`, database setup files, and Cloudflare-related setup documents. It reflects real-world delivery work beyond a simple brochure site.

### [OWS](https://github.com/Shiryu-Studios-LLC/OWS)
Open World Server work connected to multiplayer backend systems and large-world online infrastructure.

### [HubWorldMMO](https://github.com/Shiryu-Studios-LLC/HubWorldMMO)
A hub-world MMO example project tied to broader exploration around Unreal Engine, online worlds, and scalable multiplayer patterns.

### [PublicPermEx](https://github.com/Shiryu-Studios-LLC/PublicPermEx)
A public-facing repository connected to the PermissionEx surface within the Shiryu ecosystem.

## Broader Repository Mix
The organization’s public repositories also include work across Unreal plugins, experimental engines, wake-word tooling, templates, and infrastructure-adjacent projects, including repos such as **Mate-Engine**, **openWakeWord**, and **RedwoodPlugins**.

## Technology Stack
Based on the organization’s public footprint, the current stack spans multiple layers:

- **Frontend:** React, Vite, TypeScript, JavaScript, HTML/CSS
- **Edge and Backend:** Cloudflare Workers, Hono, API-driven service patterns
- **Identity:** Centralized account and authentication surfaces
- **Operations:** Internal portals, platform tooling, workflow support systems
- **Game Development:** C++, Unreal Engine-oriented repositories, multiplayer infrastructure exploration
- **Delivery:** Reusable templates, deployment docs, migrations, and environment setup workflows
